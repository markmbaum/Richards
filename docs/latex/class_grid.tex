\doxysubsection{Detailed Description}
class for constructing and containing grid information 

The model grid is one dimensional with geometrically spaced elements. The surface cell is always the smallest. Then each cell is {\ttfamily delzfrac} times larger than its shallower neighbor. The maximum cell size is set by {\ttfamily delzmax.}
